PlanIt

An application was designed to create itinerary (Transportation scedule, accomodation reservation, and activity plan). User can plan a trip and share it with friends and/or family members. This application can be used both for business travel and vacation. To introduce PlanIt, a website was created using WordPress. The idea was to upload the application through the website.
